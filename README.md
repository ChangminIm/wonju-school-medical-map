# 원주시·용인시 초중등학교-의료기관 접근성 지도

학교 주변 의료기관 접근성을 시각화하는 인터랙티브 웹 지도입니다.

## 주요 기능

### 접근성 지도 (메인 페이지)
- **원주시 / 용인시 / 전체** 탭 전환
- 학교(초·중) 및 의료기관(소아청소년과, 내과, 기타) 마커 표시
- 학교 클릭 시 가까운 의료기관 거리순 목록 + 1km/3km 반경 요약
- 의료기관 종별(상급종합병원, 병원, 의원) 필터
- 지역별 건강검진 필터 (원주: 학생건강검진 / 용인: 국가건강검진)
- 검색 기능

### 2SFCA 접근성 분석 (용인시)
- 500m 격자 기반 2SFCA / E2SFCA(가우시안 감쇠) 분석
- 캐치먼트 거리: 1km, 3km, 5km
- 공급 지표: 소아청소년과 전문의, 내과 전문의, 소아+내과, 기관 수
- KDE 밀도 분석 + Getis-Ord Gi* 핫스팟 분석
- 용인시 행정경계 오버레이
